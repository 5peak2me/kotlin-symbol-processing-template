**Kotlin Symbol Processing Template** is a repository that provides a pure template to make it easier to create a new tool project for the Kotlin Symbol Processing.(check the [Creating a repository from a template][gh:template] article).

The main goal of this template is to speed up the setup phase of ksp tools development for both new and experienced developers by preconfiguring the project scaffold and CI, linking to the proper documentation pages, and keeping everything organized.
